# Eever Sales Dashboard - Power BI

## 📌 Sobre o projeto

Este projeto foi desenvolvido como parte de um desafio de Business Intelligence da Eever, com o objetivo de analisar dados de vendas e construir um dashboard executivo no Power BI.

A proposta foi transformar uma base de dados bruta em uma visualização clara, interativa e orientada à tomada de decisão, permitindo analisar o desempenho de vendas por região, cidade, rede e produto.

## 🎯 Objetivo

Construir um dashboard em Power BI capaz de responder perguntas de negócio como:

- Qual é o total de vendas?
- Quantas cidades estão presentes na base?
- Qual é a média de vendas por rede?
- Quais estados e cidades apresentam maior volume de vendas?
- Quais produtos concentram a maior parte das vendas?
- Como as vendas se distribuem entre as redes?
- Quais produtos devem ser priorizados com base na análise de Pareto?

## 🛠️ Ferramentas utilizadas

- Power BI
- Power Query
- DAX
- Excel
- Modelagem de dados
- Visualização de dados

## 📊 Principais indicadores

Durante o desenvolvimento do projeto, foram criadas medidas e indicadores para apoiar a análise dos dados:

- Total de vendas
- Quantidade de cidades
- Média de vendas por rede
- Vendas por produto
- Vendas por estado
- Vendas por cidade
- Distribuição de vendas por rede
- Análise de Pareto por produto

## 🧠 Etapas do projeto

O projeto foi desenvolvido seguindo as principais etapas de um fluxo de análise de dados:

1. Importação da base de dados no Power BI
2. Tratamento e padronização dos dados no Power Query
3. Ajuste dos tipos de dados
4. Criação de medidas DAX
5. Desenvolvimento dos indicadores executivos
6. Construção dos visuais analíticos
7. Aplicação de filtros interativos por estado e cidade
8. Construção do layout do dashboard
9. Organização da narrativa visual para facilitar a interpretação dos dados

## 📈 Visuais desenvolvidos

O dashboard foi estruturado com diferentes tipos de visualizações para facilitar a análise:

- Cards com indicadores principais
- Mapa geográfico com vendas por região
- Gráfico de linha para análise percentual de vendas
- Gráfico de rosca para distribuição de vendas
- Tabela analítica
- Gráfico de Pareto para análise dos principais produtos
- Filtros interativos por Estado e Cidade
